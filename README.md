Sử dụng: NodeJS, ExpressJS, TypeScripts, GraphQL.
Đường dẫn: article-graphql.vercel.app/graphql để query.

Article:
- getListArticle
- getArticle
- createArticle
- updateArticle
- deleteArticle

Category:
- getListCategory
- getCategory
- createCategory
- updateCategory
- deleteCategory

User:
- getUser
- registerUser
- loginUser
